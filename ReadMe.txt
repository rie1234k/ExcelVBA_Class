ExcelVBA クラスモジュール関連 各フォルダの説明

《BookCheck》フォルダ
エクセルブック操作用クラスモジュール
プロパティ・メソッドの詳細はBookChek_Sample.xlsmに記載

《CalenderForm》フォルダ
カレンダーフォーム用クラスモジュール・ユーザーフォーム

《ControlEvents》フォルダ
ユーザーフォームのコントロールのイベントを共通化して設定する

《FormSet》フォルダ
ユーザーフォーム設定用クラスモジュール
ユーザーフォームの表示位置の設定、デザインの統一を行う

《Picker》フォルダ
ファイル・フォルダ取得用クラスモジュール


《ProgressBar》フォルダ
プログレスバー表示用クラスモジュール・ユーザーフォーム
